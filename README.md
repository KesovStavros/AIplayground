# AIplayground
Игра песочница, в которой игрок имеет возможность взаимодействовать с ИИ.<br>
Объекты:<br>
1 - кресло,<br>
2 - телевизор,<br>
3 - болгарка,<br>
4 - ящик.<br>
Ящик позволяет предметам и телу ИИ не падать.<br>
Для выбора предмет нажмите соответствующий их номеру клавишу.<br>
Другие клавиши:<br>
s - сохранить ИИ<br>
l - загрузить ИИ<br>
r - удалить предмет, на котором находится курсор.<br>
При запуске вас попросят выбрать количество эпох - периодов обучения
