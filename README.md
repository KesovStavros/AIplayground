# AIplayground
Игра песочница, в которой игрок имеет возможность взаимодействовать с ИИ.
Объекты:
1 - кресло,
2 - телевизор,
3 - болгарка,
4 - ящик.
Ящик позволяет предметам и телу ИИ не падать.
Для выбора предмет нажмите соответствующий их номеру клавишу.
Другие клавиши:
s - сохранить ИИ
l - загрузить ИИ
r - удалить предмет, на котором находится курсор.
При запуске вас попросят выбрать количество эпох - периодов обучения
